# Prototype Pain: Compats

This is a mod to add compatibility to mods that otherwise would be weird/nonfunctional to use with [Prototype: Pain](https://www.curseforge.com/minecraft/mc-mods/prototype-pain)

## Currently Supported Mods:

<details>
<summary>Cuffed</summary>


- Hanged players will now suffocate and have their neck snapped if they fall fast enough
- Being restrained on the arms disables the ability to do health things

</details>

<details>
<summary>Create</summary>

- Crushing wheels cause massive blunt damage and always fractures random limbs per hit

</details>

<details>
<summary>Siren Head: The Arrival</summary>

- Siren Head when crawling now deals massive limb damage, can amputate limbs, and can remove eyes.
- Siren Head normally does heavy blunt damage with stomps and slams, and causes hearing loss + brain damage with their sonic boom

</details>

## Suggesting a Mod

If you want support for a specfic mod, create an issue with the mod and how it would function (do note that changes may or may not be made to how it would function)
